User Guide

Welcome to the official EloScouter client! EloScouter is a robotics prediction application.

How it works

Here's the short version:
You feed the program data from previous matches in the competition.
Then, the program will extract all the data and create a prediction for every category.
Using these predictions, it will rank each team.
When a match is predicted, it will predict the win/loss using these rankings.

How to use

Step 1. Import CSV file

To begin, click on the "Import CSV file" button to give the program your scouting data.
Learn how to download Google Sheets as CSV.

Select your desired CSV file using the file dialog.

Step 2. Match the categories with the columns in the CSV file

Using the column selection dialog, match the preset categories with the columns in your CSV file.
The program will automatically parse your file and extract the CSV columns.

After you select the columns, it will be saved in cache in a {filename}.elocache file.
The program will be able to auto-fill the columns next time you import this CSV file.
If you do not wish for this to happen, simply delete the {filename}.elocache file.

The preset columns may be modified within the code, when the competition changes.
This can be done in PredictScouter.columns.columns.

Step 3. Predict a match

Once you have successfully imported the CSV file, press "Predict a match" to predict a match.
At this point, you can also change the CSV file and start again by pressing "Import CSV file" (return to Step 1).

Use the "+" to increase a team, and use the dropdown selections to select a team.
The left side is red alliance, and the left side is blue alliance.
To remove a team, just leave the dropdown selection blank.

Step 4. View results

At the top of the results window, it will display the winning alliance.
Below that, the window will display the teams of the red and blue alliance (separated) in a table.
Within each table (alliance), the teams will be sorted from best (top) to worst (bottom).
The red alliance table will always appear above the blue alliance table.
You can view the specific predicted results for each category for each team.
